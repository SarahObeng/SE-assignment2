# SE-assignment2


Project Description

This project was completed for COMP 3322 (Software Engineering) to practice using Git and GitHub for version control. The assignment focused on creating a repository, making commits, working with branches, and using pull requests.

The program is a simple Python script that originally printed "Hello, World!" and was later updated to include a user’s name and additional output.

Installation Instructions

Make sure Python is installed on your computer

Clone the repository:

git clone https://github.com/SarahObeng/SE-assignment2.git

Go into the project folder:

cd SE-assignment2
Usage

Run the program with:

hello_world_program(2).py

Example Output:
Hello, World!
Hello Sarah
Afrah World!
Obeng World!
Features

Basic "Hello, World!" program

Added personalized greeting using a name variable

Demonstrates use of branches and merging in GitHub

Additional Notes

feature-1 branch was used to add the greeting feature

feature-2 branch was used to create and resolve a merge conflict

Merge conflict occurred when different name values were used and was resolved by creating new variables and output lines

GitHub Issues were used to track adding more name variables and improving code comments

What I Learned

How to create and manage branches in Git

How pull requests work for reviewing and merging code

How to resolve merge conflicts manually

How GitHub Issues can help organize tasks
